# StampTourWithAniSaniParki
- "안산 산업 역사 박물관"의 전시효과 상승을 목표로 진행된 프로젝트입니다.
- 박문관 내부에 QR 마커가 전시된다는 가정하에 내용이 구성되어 있으며 내부 콘텐츠의 권장 진행순서는 다음과 같습니다.
- [지번안내도] -> [3D 재구성] -> [흑백/컬러 TV] -> [색칠놀이] -> [사진촬영]
